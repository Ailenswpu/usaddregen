# Store Listing Draft

## Short Description

Generate synthetic US and international address test data for QA, demos, and form development.

## Full Description

US Address Generator helps developers, QA engineers, designers, and support teams create realistic-format placeholder address data without leaving the browser.

Use the popup to generate a random address, copy it, or fill common address fields on the current page. The extension shares the same lightweight static datasets used by usaddregen.com, including all 50 US states plus Nigeria, Egypt, Turkey, and Pakistan.

Built for:

- Checkout and address-form testing
- QA scripts and manual testing
- UI demos and product screenshots
- Seed data and mock customer records
- Documentation examples

Compliance note:

Generated records are synthetic and realistic-format only. They are not deliverable and are not meant to pass postal, AVS, banking, telecom, government, shipping, KYC, or identity verification. Do not use this extension for fraud, account abuse, identity claims, rule evasion, or illegal activity.
